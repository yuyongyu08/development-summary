# 修饰符

| 区别                 | var | let | const |
| ------------------ | --- | --- | ----- |
| 是否有块状作用域           | ❌   | ✅   | ✅     |
| 是否存在变量提升           | ✅   | ❌   | ❌     |
| 能否添加全局变量           | ✅   | ❌   | ❌     |
| 能否重复声明             | ✅   | ❌   | ❌     |
| 是否存在暂时性死区（声明之前不可用） | ❌   | ✅   | ✅     |
| 是否必须设置初始值          | ❌   | ❌   | ✅     |
| 能否改变指针             | ✅   | ✅   | ❌     |
