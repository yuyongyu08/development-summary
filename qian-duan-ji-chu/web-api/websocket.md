# WebSocket



| 区别    | WebSocket    | Ajax    |
| ----- | ------------ | ------- |
| 基于的协议 | TCP          | HTTP    |
| 生命周期  | 长连接          | 短连接     |
| 发起人   | 客户端、服务端都可以   | 客户端     |
| 适用范围  | 实时数据交互（消息推送） | 非实时数据交互 |



参考：

* [https://developer.mozilla.org/en-US/docs/Web/API/WebSockets\_API](https://developer.mozilla.org/en-US/docs/Web/API/WebSockets\_API)
