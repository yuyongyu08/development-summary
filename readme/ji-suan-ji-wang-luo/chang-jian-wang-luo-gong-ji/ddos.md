# DDos

分散式阻断服务攻击（**D**istributed **D**enial-**o**f-**s**ervice attack，简称DDoS攻击）



通过合法方式或伪造方式海量向目标服务器发起攻击，意图通过抢占资源，导致正常用户无法访问
